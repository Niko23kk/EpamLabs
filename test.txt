2. Добавлении товара в избранное
	Шаги:
		Перейти по ссылке https://vans.ru/catalog/item/41979-kedy-flame-old-skool.html->
		Нажать на кнопку "Добавить в избранное"

	Ожидаемый результат: В headere в span элементе с классом js-wishlist-count появится значение 1

3. Фильрация товаров по ценовомову диапозону
	Шаги:
		Перейти по ссылке https://vans.ru/catalog/men/-> 
		Поставить минимальную цену равную 10080

	Ожидаемый результат: Должны отобразиться товары у которых свойство data-impression-data-price>=10080

4. Фильтрация товаров по цвету
	Шаги:	
		Перейти по ссылке https://vans.ru/catalog/men/-> 
		Выбрать цвет бежевый

	Ожидаемый результат: Отображаются товары у которых свойство data-impression-data-variant содержит слово "Бежевый"
	
5. Выбор разных цветов товаров
	Шаги:

6. Оформление заказа при оплате через карту
	Шаги: 
		Перейти по ссылке https://vans.ru/catalog/item/41979-kedy-flame-old-skool.html-> 
		Выбрать функицю добавить в корзину->
		Выбрать "Оформить заказ"->
		Выбрать формат оплаты через карту

	Ожидаемый результат: На панели с классом "b-checkout-new__aside" появляется скидка в рублях 5% от начальной цены, уменьшается итоговая сумма на 5% от начальной цены
